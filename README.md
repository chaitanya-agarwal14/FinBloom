# FinBloom: Knowledge Grounding Large Langage Model with Real-Time Financial Data

Finbloom is a framework designed to help Large Language Models answer a wide variety of financial queries by retrieving relevant financial data required to answer a given financial query.

## Features

- **Financial Context Dataset**: The Financial Context Dataset is a custom-built collection of over 50,000 natural language user queries mapped to structured data requirements for seamless retrieval via data modules. Developed through a combination of direct retail customer interactions across digital channels and expert consultations with financial advisors and investors, this comprehensive dataset bridges the gap between complex financial inquiries and actionable data structures. More details about the dataset can be found at the [kaggle page](https://www.kaggle.com/datasets/ankurzing/financial-context-dataset) for the dataset.
